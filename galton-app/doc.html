Introduction
This app ( source code in Github ) is created using RStudio's Shiny package as a part of an assignment in Coursera MOOC's Developing Data Products. It predicts child's height at adult age from parents height(s) and child's gender.
The model is fitted from data based on the famous 1885 study by Sir Francis Galton, exploring the relationship between the heights of adult children and the heights of their parents. The fitted linear model is quite accurate and it was claimed that the model can challenge modern genomic studies.
Input
User must select or enter values for these parameters:
Prediction based on: The prediction is based on both parents' heights or mother's or father's height only.
Father's Height in cm: Use this slider to select father's height in cm.
Mother's Height in cm: Use this slider to select mother's height in cm.
Child's Gender: Select gender of the child
Confidence Interval % : Select confidence level for prediction interval, can be from more than 0 to less than 100. Widely accepted default value is 95%. The prediction interval tells us about the distribution of values. For example with confidence level of 95% we expect that 95% of the samples to lie within this interval.
Result
The app gives the predicted adult child's height in cm, as well as prediction interval of the given confidence level.
Comparison Plot Comparison plot shows child's height compared to father's and/or mother's height(s). The darker box on top of child's bar represents prediction interval, so top of this box equals to upper limit of prediction interval, and bottom - lower limit. The horizontal red line with number represents predicted child's height.
